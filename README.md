# directus-template
Directus template

# Инструкция по запуску проекта

## Предварительные требования
Убедитесь, что на вашем компьютере установлены:
- Git
- Node.js (версия указана в `package.json` и `.nvmrc`)
- npm

## Шаги по установке и запуску

### 1. Клонирование репозитория
```bash
git clone git@github.com:misterEugene/directus-template.git
```

### 2. Переход в директорию проекта
```bash
cd directus-template
```

### 3. Установка зависимостей
```bash
npm i
```

### 4. Настройка переменных окружения

Скопируйте файл example.env в корневую директорию проекта и переименуйте его в `.env` вручную или используйте команду:
```bash
cp env.example .env
```

Заполните в файл `.env` значения переменных окружения согласно подсказками из квадратных скобок.

### 5. Создание системных таблиц директуса в бд
```bash
npm run bootstrap
```

### 6. Запуск проекта
```bash
npm run start
```

Если всё сделать правильно, после выполнения этой команды проект должен быть доступен локально по адресу `http://[YOUR_DIRECTUS_HOST]:[YOUR_DIRECTUS_PORT]`
